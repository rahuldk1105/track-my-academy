### 1. Set Up MongoDB Atlas (5 minutes)
Follow the guide in `mongodb-atlas-setup.md`:
1. Create MongoDB Atlas account
2. Create free cluster named "track-my-academy-prod"
3. Set up database user and network access
4. Get connection string

### 2. Configure Render Environment Variables (5 minutes)
1. Go to your Render dashboard
2. Add all backend environment variables listed above
3. **Important**: Replace `MONGO_URL` with your actual MongoDB Atlas connection string

### 3. Configure Vercel Environment Variables (3 minutes)
1. Go to your Vercel dashboard
2. Add all frontend environment variables listed above
3. **Important**: Update `REACT_APP_BACKEND_URL` once your Render backend is deployed

### 4. Update Supabase Settings (2 minutes)
In your Supabase dashboard > Authentication > Settings:
- **Site URL**: `https://trackmyacademy.vercel.app`
- **Redirect URLs**: Add `https://trackmyacademy.vercel.app/**`

### 5. Deploy and Test (5 minutes)
1. Both platforms should auto-deploy when you push to your connected repositories
2. Test backend health: `https://your-backend-url.onrender.com/api/health`
3. Test frontend: `https://trackmyacademy.vercel.app`
4. Create super admin: Visit `https://your-backend-url.onrender.com/api/create-super-admin`
